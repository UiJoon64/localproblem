### 2. 데이터 전처리

2-1. 연도별 고령 인구 결측치 처리
 <1-1>의 데이터를 처리할 때, 15년간 각 행정동의 명칭이 변화(소멸, 병합, 분할, 승강)함에 따라 결측란이 발생. 각각의 경우에 따라 인구 결측란을 서로 다른 방법으로 처리
 2-1-1. 소멸의 경우

 2-1-2. 병합의 경우

  해당 구역이 병합된 경우

 2-1-3. 분할의 경우

  해당 구역이 분할된 경우

 2-1-4. 승강의 경우

  해당 구역의 승격, 혹은 강등이 된 경우
